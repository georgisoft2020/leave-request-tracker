## :exclamation: Run examples locally
 - #### on Windows:
`
  git clone https://github.com/scavasoft/leave-request-tracker.git
  cd frontend
  npm install
  npm run start
  cd..
  cd backend
  npm install
  nodemon app.js
`
 - #### on Linux:
 `
   git clone https://github.com/scavasoft/leave-request-tracker.git
   npm install
   on root directory 'npm run dev' 
 `
